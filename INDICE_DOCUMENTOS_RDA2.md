2. Click derecho en archivo .md → "Markdown PDF: Convert"
3. Se genera .pdf automáticamente

### Opción 3: Pandoc (Línea de comandos, profesional)

```bash
# Instala Pandoc
brew install pandoc

# Convierte Markdown a PDF con estilos profesionales
pandoc INFORME_OPTIMIZACION_RDA2_CRITERIO2.md \
  -o INFORME_RDA2_CRITERIO2.pdf \
  --pdf-engine=xelatex \
  --template=eisvogel \
  -V geometry:margin=2.5cm \
  -V fontsize=11pt
```
